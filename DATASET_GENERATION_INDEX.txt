USING THE DATASET

BASIC LOADING:

```python
import cv2
import pandas as pd
from pathlib import Path

# Load annotations
annotations = pd.read_csv("dataset_ir_realistic/annotations.csv")

# Load an image
img = cv2.imread("dataset_ir_realistic/images/img_000000.png", 
                 cv2.IMREAD_GRAYSCALE)

# Access features
row = annotations[annotations['filename'] == 'img_000000.png'].iloc[0]
print(f"Label: {row['label']}")  # 1, 0, or -1
print(f"Mean Intensity: {row['mean_intensity']}")
```

# Easy samples for Tier 1
