## FEATURE ENGINEERING REFERENCE

### 17 Features & Their Roles

```
????????????????????????????????????????????????????????????????
? FEATURE IMPORTANCE RANKING                                  ?
????????????????????????????????????????????????????????????????

TOP 5 MOST IMPORTANT:
1. Circularity
   ?? Bullets are circular
   ?? Non-bullets are irregular
   ?? Range: [0, 1]
   ?? Easy to interpret

2. Area
   ?? Bullet size distinctive
   ?? Range: [0, ?)
   ?? After normalization: ? ? 0, ? ? 1

3. Solidity
   ?? Filling of contour
   ?? Bullets are solid
   ?? Range: [0, 1]

4. Hu Moments (especially 1st & 2nd)
   ?? Scale & rotation invariant
   ?? Capture shape characteristics

5. Aspect Ratio
   ?? Width / Height
   ?? Bullets near 1.0
   ?? Non-bullets vary widely

SUPPORTING FEATURES:
6-17. Other Hu moments, channels, moments
      ?? Provide context and redundancy
      ?? Prevent overfitting to single features
```

### Normalization Formula

```
FOR each feature i:
    x_normalized[i] = (x_raw[i] - mean[i]) / std_dev[i]

Stored in: data/models/scaler.bin

Example:
    mean[0] = 5000.0 (avg area in training data)
    std_dev[0] = 2000.0
    
    Raw value: 7000
    Normalized: (7000 - 5000) / 2000 = 1.0
    
This ensures all features have mean ? 0 and std ? 1
```

## THRESHOLD REFERENCE

### Decision Thresholds

```
??????????????????????????????????????????????????
? TIER 1 (Naive Bayes) THRESHOLD                ?
??????????????????????????????????????????????????
? Threshold: 0.5                                ?
? Action if confidence > 0.5:                   ?
?   ?? Pass to Tier 2 for refinement            ?
? Action if confidence ? 0.5:                   ?
?   ?? Reject (not a candidate)                 ?
?                                               ?
? Purpose: High recall (catch all candidates)  ?
? False Positive Rate: ~10% (OK, filtered later)?
??????????????????????????????????????????????????

??????????????????????????????????????????????????
? TIER 2 (MLP) THRESHOLD                        ?
??????????????????????????????????????????????????
? Threshold: 0.7                                ?
? Action if confidence > 0.7:                   ?
?   ?? ACCEPT (bullet hole detected!)           ?
? Action if confidence ? 0.7:                   ?
?   ?? Reject (false positive)                  ?
?                                               ?
? Purpose: High precision (eliminate FP)        ?
? False Positive Rate: ~5% (final output)       ?
??????????????????????????????????????????????????

??????????????????????????????????????????????????
? HARD SAMPLE DETECTION THRESHOLD               ?
??????????????????????????????????????????????????
? Sample is HARD if ANY:                        ?
?  1. Misclassified: pred_label ? true_label    ?
?  2. Uncertain: |conf - 0.5| < 0.15            ?
?  3. Boundary: margin < 0.1                    ?
?                                               ?
? Typically 20-30% of dataset detected as hard  ?
? These samples prioritized for augmentation    ?
??????????????????????????????????????????????????

??????????????????????????????????????????????????
? CONVERGENCE THRESHOLD                         ?
??????????????????????????????????????????????????
? Stop if:                                      ?
?  accuracy_improvement < 0.001 (0.1%)          ?
?  for 3 consecutive iterations                 ?
?                                               ?
? Typical convergence: 8 iterations / 10 max    ?
??????????????????????????????????????????????????

??????????????????????????????????????????????????
? STABILITY CHECK THRESHOLD                     ?
??????????????????????????????????????????????????
? Allowed performance drop: 5%                  ?
?                                               ?
? IF old_accuracy_drop > 5%:                    ?
?   ?? WARNING: Catastrophic forgetting         ?
```

## TROUBLESHOOTING QUICK REFERENCE

### Common Issues

```
ISSUE: "Models file not found"
FIX:   Check data/models/ directory
       Verify tier1_classifier.bin, tier2_mlp.bin exist
       Run with correct model_dir path

ISSUE: "CSV parsing errors"
FIX:   Ensure 17 features + 1 label per line
       Check for NaN values
       Validate feature ranges

ISSUE: "Accuracy drops after training"
FIX:   Increase replay_buffer_ratio
       Check convergence curve (shouldn't drop)
       Verify stability check results

ISSUE: "Takes too long to train"
FIX:   This is expected (~50-60 seconds for incremental)
       Much faster than full retrain (4+ hours)
       Timing breakdown available in logs/

ISSUE: "Training doesn't converge"
FIX:   Check if 10 iterations is enough
       Increase max_iterations parameter
       Verify dataset quality
```
