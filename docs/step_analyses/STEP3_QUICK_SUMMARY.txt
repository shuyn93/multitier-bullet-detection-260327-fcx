## Usage Pattern

```cpp
// 1. Create config
FeatureConfig config;
config.radial_ray_count = 32;
config.use_standardization = true;

// 2. Extract training features
ImprovedFeatureExtractor extractor(config);
std::vector<std::vector<float>> train_feat;
for (each training sample) {
    auto feat = extractor.extractFeatures(roi, contour);
    train_feat.push_back(feat.toVector());
}

// 3. Compute training statistics
FeatureStatistics stats;
stats.computeStatistics(train_feat);

// 4. Extract test features with standardization
auto test_feat = extractor.extractFeatures(
    test_roi, test_contour, &stats
);
```
