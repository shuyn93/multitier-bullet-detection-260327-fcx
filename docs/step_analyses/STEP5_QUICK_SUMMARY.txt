## Escalation Logic

**Smart margin-based decision**:
- High margin (0.3+) = confident, don't escalate
- Medium margin (0.1-0.3) = escalate to next tier
- Low margin (<0.1) = strongly escalate
- Weak rejects can also escalate (if enabled)

**Example**:
- Confidence 0.75 ? margin 0.25 ? ESCALATE (on boundary)
- Confidence 0.85 ? margin 0.35 ? DON'T ESCALATE (confident)
- Confidence 0.99 ? margin 0.49 ? EARLY TERMINATION (very confident)

## Tier 3 SVM Confidence

### Before: Forced
```cpp
if (majority_vote) {
    confidence = 0.95f;  // ? Forced high
} else {
    confidence = 0.10f;  // ? Forced low
}
```

### After: From Votes
```cpp
// 3 SVMs
int accept_votes = number_voting_accept;
float confidence = accept_votes / 3.0f;

// 0 votes (0.0) ? confidence 0.0 (REJECT)
// 1 vote (0.33) ? confidence 0.33 (REJECT, tight)
// 2 votes (0.67) ? confidence 0.67 (ACCEPT, tight)
// 3 votes (1.0) ? confidence 1.0 (ACCEPT, unanimous)
```

**Benefit**: Realistic confidence reflects vote decisiveness
