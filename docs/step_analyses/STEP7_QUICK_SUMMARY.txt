## M-Estimator Weighting

**Tukey Biweight**:
```
normalized = |residual| / scale
if normalized < 4.685:
    weight = (1 - (normalized/4.685)^2)^2
else:
    weight = 0  (complete downweighting)
```

**Scale Estimation (MAD)**:
```
median_absolute_deviation = median(|r - median(r)|)
scale = MAD / 0.6745  (normal distribution factor)
```
