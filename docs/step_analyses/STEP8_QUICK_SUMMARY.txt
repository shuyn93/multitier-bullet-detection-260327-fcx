## Kalman Filter Features

### State Representation
```
6-D state: [x, y, z, vx, vy, vz]
- Position: 3D location
- Velocity: motion direction & speed
```

### Optimal Estimation
```
? Minimizes mean square error
? Tracks uncertainty
? Principled parameter tuning
? Multi-step prediction
```

### Covariance Tracking
```
P[6×6]   - State uncertainty
Q[6×6]   - Process noise model
R[3×3]   - Measurement noise
```

## Track Association Improvements

### Mahalanobis Distance
```
Traditional: Euclidean distance
New: Mahalanobis (accounts for covariance)

d_M = sqrt(residual^T * S_inv * residual)

Advantages:
? Statistical basis
? Automatic gating
? Handles uncertainty
```

### Hungarian Algorithm
```
Before: Greedy (order-dependent)
After: Optimal global assignment

Benefits:
? Optimal assignment
? No track swaps
? Handles ambiguities
```
